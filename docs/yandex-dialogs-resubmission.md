# Yandex Dialogs resubmission checklist

## Почему модерация отклонила навык

Модератор указал две проблемы:

- описание навыка не объясняет функции навыка и его создателя;
- при проверке кнопок навык открыл авторизацию, но введенные данные дали ошибку
  "Неверный email или пароль".

У навыка есть account linking, поэтому в заметках для модератора нужно явно
описать авторизацию и дать рабочие тестовые данные. OAuth-страница также
поддерживает создание нового аккаунта: если тестовый аккаунт не подготовлен,
модератор может нажать "Создать и связать".

## Описание навыка

Текст для поля "Описание":

```text
Chaotika — навык для голосового добавления задач и покупок в одноименный планер. После связки аккаунта пользователь может сказать "добавь задачу позвонить завтра в 9" или "надо купить молоко", а навык сохранит задачу или покупку в личном рабочем пространстве Chaotika.

Создатель: <имя или название владельца навыка>. Контакт для вопросов по работе навыка: <email>.
```

Перед отправкой замените плейсхолдеры на реальные данные владельца и рабочий
email для связи.

## Заметки для модератора

Текст для поля "Заметки для модератора":

```text
Навык использует связку аккаунта Chaotika. Для запуска авторизации скажите или нажмите пример: "добавь задачу купить молоко". Навык покажет карточку авторизации.

Для проверки входа:
Логин: <test-email>
Пароль: <test-password>

Если тестовый аккаунт не найден, на странице авторизации можно указать email, имя и пароль от 6 символов, затем нажать "Создать и связать". После успешной связки навык умеет:
- "добавь задачу позвонить завтра в 9" — создает задачу;
- "надо купить молоко" — добавляет покупку в список покупок;
- "прочитай задачи на сегодня" — зачитывает активные задачи на сегодня;
- "помощь" — показывает примеры команд.
```

Проверьте тестовый логин и пароль на production перед повторной отправкой.

## Smoke test

```bash
curl -fsS https://chaotika.ru/api/health
```

```bash
curl -fsS https://chaotika.ru/api/v1/alice/webhook \
  -H 'Content-Type: application/json' \
  -d '{"meta":{"interfaces":{"account_linking":{}},"timezone":"Europe/Moscow"},"request":{"command":"добавь задачу купить молоко","type":"SimpleUtterance"},"session":{"new":false},"version":"1.0"}'
```

Ожидаемый ответ без авторизации:

```json
{
  "start_account_linking": {},
  "version": "1.0"
}
```
